# 超运费采购单分析功能说明

## 🎯 功能概述

在 SellFox 采购管理页面中，本插件可以自动识别"待到货"状态下的"单据"表格，并提供超运费采购单的分析功能。

## 📍 功能触发条件

插件会在以下条件下自动注入分析按钮：

### 1. URL 要求
```
https://www.sellfox.com/amzup-web-main/amzup-web-vue3?amzup-web-vue3=%2Fvue3%2Fweb%2Fv3%2FpurchaseManage%2Findex.html
```

### 2. 页面状态要求
- ✅ "待到货" Tab 处于激活状态（`li.is-active` 包含 `span.main-content.text_ellipsis` 且文本包含"待到货"）
- ✅ "单据" 按钮处于激活状态（`label.is-active` 包含 `span.el-radio-button__inner` 且文本包含"单据"）

### 3. DOM 结构验证
```html
<!-- 待到货 Tab -->
<li class="is-active">
  <span class="main-content text_ellipsis" title="">待到货</span>
</li>

<!-- 单据按钮 -->
<label class="is-active">
  <span class="el-radio-button__inner">单据</span>
</label>
```

## 🚀 功能使用方法

### 1. 自动注入
当页面满足上述条件时，插件会自动在以下位置注入分析按钮：

**按钮位置**：`div.main_header_container > div.screen_l > div.sf_batch_btn_bar` 后面

### 2. 按钮样式
- 按钮文本：**取消超运费采购单**
- 按钮颜色：红色到橙色渐变（`#f56c6c` → `#e6a23c`）
- 悬停效果：向上移动 2px 并增强阴影

### 3. 点击功能
点击按钮后，插件会：

1. **扫描表格**：读取 `div.vxe-table--body-inner-wrapper` 中的所有行数据
2. **提取数据**：从每行中提取以下列的数值：
   - `col_66`：运费金额
   - `col_68`：采购金额
   - `col_51`：单据号
3. **计算比率**：计算运费/采购金额的比率
4. **过滤结果**：筛选出比率 > 1 的采购单
5. **展示结果**：以弹窗表格形式展示超运费采购单

## 📊 数据提取逻辑

### 表格行识别
- 每行通过 `div[rowid="row_*"]` 识别，`*` 为数字（非连续）
- 插件会自动扫描表格中存在的所有 rowid

### 数值提取
```javascript
// 从文本中提取数字（支持小数）
"¥123.45" → 123.45
"67.89元" → 67.89
"100" → 100
```

### 计算公式
```javascript
运费比率 = col_66数值 / col_68数值
```

## 🔧 技术实现细节

### 自动检测机制
插件使用 `MutationObserver` 监听页面变化，当检测到：
1. URL 包含特定路径
2. 页面 DOM 满足激活状态条件
3. 按钮插入位置存在

自动触发按钮注入。

### 数据提取流程
```javascript
1. 查找表格容器 div.vxe-table--body-inner-wrapper
2. 获取所有行 div[rowid^="row_"]
3. 对每行提取 div[colid="col_66/68/51"]
4. 数值清理和计算
5. 比率过滤 (ratio > 1)
6. 结果展示
```

### 错误处理
- ✅ 表格不存在时显示提示
- ✅ 列数据缺失时跳过该行并记录日志
- ✅ 无超运费数据时显示友好提示

## 📝 使用注意事项

1. **页面状态**：确保在"待到货" + "单据"状态下使用
2. **数据刷新**：页面数据更新后需重新点击按钮
3. **弹窗关闭**：点击关闭按钮或弹窗外部区域可关闭
4. **计算精度**：支持小数点后两位精度

### 常见问题排查

**问题**：按钮不出现
- 检查 URL 是否包含 `purchaseManage/index.html`
- 确认"待到货"和"单据"是否激活
- 查看 Console 日志确认注入状态

**问题**：分析结果为空
- 确认表格数据是否加载完成
- 检查运费/金额列是否包含有效数值
- 查看提取的原始数据验证计算逻辑
